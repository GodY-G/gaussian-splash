### §3.1 概率基础（demo_01）

#### 一、PDF vs CDF

**概率密度函数 PDF** `p(x)`：
- 描述随机变量在某个值附近出现的"可能性密度"
- **不是概率本身**——`p(x)` 可能 >1，但积分 =1
- 概率 = PDF 在区间上的积分：`P(a<X<b) = ∫[a,b] p(x)dx`

**累积分布函数 CDF** `F(x) = P(X≤x)`：
- x 处的 CDF 值 = PDF 从 -∞ 到 x 的积分
- CDF 单调递增，从 0→1

demo 演示了三种分布：

| 分布 | PDF 特征 | CDF 特征 | 3DGS 关联 |
|------|---------|---------|----------|
| **正态** N(0,1) | 钟形，中心对称 | S 型曲线 | **核心：3DGS 的高斯基元就是 3D 高斯** |
| **均匀** U(-2,2) | 常数（区间内） | 线性增长 | 从不使用——高斯比均匀更"平滑可微" |
| **指数** Exp(1) | 单调递减 | 渐近 1 | 不透明度衰减的参考模型 |

#### 二、期望与方差的几何意义

```
期望 μ = 分布的"重心"（中心位置，分布围绕它对称）
       → 改变 μ → PDF 整体平移，形状不变

方差 σ² = 分布的"宽度"（分散程度）
       → 改变 σ → PDF 变窄/变宽，中心不变
```

**3DGS 中**：每个高斯基元有 3D 位置 μ（中心）和协方差矩阵 Σ（形状+方向）。

#### 三、为什么 3DGS 用高斯而非其他分布

1. **可微性**：高斯函数处处可微 → 能做梯度下降
2. **紧凑支撑**：99.7% 的质量在 μ±3σ 内 → 便于裁剪
3. **闭式积分**：高斯在任意区间上的积分有解析式（erf 函数）
4. **仿射变换保持高斯性**：高斯的线性变换仍是高斯

### §3.2 一维高斯分布（demo_02）

#### 一、数学形式

$$N(x; \mu, \sigma) = \frac{1}{\sigma\sqrt{2\pi}} \exp\left(-\frac{(x-\mu)^2}{2\sigma^2}\right)$$

代入代码（`gaussian()` 函数）：
```python
np.exp(-0.5 * ((x - mu) / sigma)**2) / (sigma * np.sqrt(2 * np.pi))
```

- `(x-μ)/σ`：**标准化**，把 x 转化为"离均值几个标准差"
- `exp(-0.5*z²)`：核心钟形曲线，z=0 时=1，z↑时间→0
- `1/(σ√2π)`：归一化系数，保证总积分为 1

#### 二、参数 μ 和 σ 的作用

```
μ（均值）→ 控制"中心位置"
  μ = -2：钟形曲线峰值在 x=-2
  μ =  0：峰值在 x=0
  μ = +3：峰值在 x=+3
  改变 μ → 分布平移（shape 不变）

σ（标准差）→ 控制"宽窄/扁平程度"
  σ = 0.5：窄而高（数据集中在 μ 附近）
  σ = 1.0：标准宽度
  σ = 3.0：宽而扁（数据散布范围大）
  改变 σ → 分布缩放（总面积始终=1）
```

#### 三、68-95-99.7 规则（三西格玛规则）

| 区间 | 包含概率 | 在 3DGS 中的意义 |
|------|:------:|------|
| μ ± 1σ | **68.27%** | 高斯基元的"核心区" |
| μ ± 2σ | **95.45%** | 绝大部分贡献在此范围内 |
| μ ± 3σ | **99.73%** | **裁剪阈值**——超出 3σ 的贡献可忽略 |

代码中的 `text` 标注（line 102-105）直接说明：

> 3DGS中的意义：高斯基元的"影响范围"通常取3σ以内，超出范围的贡献可忽略

**3DGS 中直接应用**：在 CUDA 光栅化核函数中，每个高斯只计算其 3σ 范围内像素的贡献，大幅减少计算量。

#### 四、高斯函数的**可微性**——梯度可视化

这是 3DGS 能训练的根本原因。

**对 μ 的梯度**：
$$\frac{\partial N}{\partial \mu} = N(x) \cdot \frac{x-\mu}{\sigma^2}$$

- x > μ 时：梯度为正（增大 μ 让高斯靠近 x → 概率增大）
- x < μ 时：梯度为负
- x = μ 时：梯度为 0（峰值处）
- 梯度呈**奇函数**形状

**对 σ 的梯度**：
$$\frac{\partial N}{\partial \sigma} = N(x) \cdot \left(\frac{(x-\mu)^2}{\sigma^3} - \frac{1}{\sigma}\right)$$

- |x-μ| = σ 处：梯度 = 0（拐点）
- |x-μ| < σ 时：梯度为负（缩小 σ 让分布更高更窄）
- |x-μ| > σ 时：梯度为正（增大 σ 让分布更宽）

**3DGS 中的具体对应**：

| 一维高斯 | 3DGS |
|---------|------|
| μ（均值） | 位置 `xyz`（3D 坐标）— 可学习 |
| σ（标准差）| 协方差矩阵 Σ 的对角缩放分量 `scale` — 可学习 |
| ∂N/∂μ | 位置梯度 `dL_dmeans` — 驱动高斯移动 |
| ∂N/∂σ | 缩放梯度 `dL_dscales` — 驱动高斯变大/变小 |
| N(x) 函数 | 3DGS 中的高斯响应 `G(X) = exp(-0.5*(X-μ)ᵀΣ⁻¹(X-μ))` |

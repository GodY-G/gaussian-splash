# 板块三详解：向量——点积与叉积

## 3.1 先理清：什么是向量？

你已经会在 Python 中写 `np.array([x, y, z])`。但这只是向量的**数据表示**。向量本身到底是什么？

**向量的两种视角**：

| 视角 | 理解 | 例子 |
|------|------|------|
| **几何** | 空间中一个有方向、有长度的箭头 | 从原点指向点 `(2, 3, 5)` |
| **代数** | 一组有序数 `(x, y, z)` | `np.array([2, 3, 5])` |

在 3DGS 中，每个高斯基元的**位置 `μ`** 就是一个 3D 向量：
```python
mu = np.array([1.5, 0.3, 4.2])   # 椭球中心在空间中的位置
```

## 3.2 向量的长度（模）

```python
v = np.array([3, 4])
norm = np.linalg.norm(v)    # = 5.0
```

公式就是勾股定理：`|v| = √(3² + 4²) = √(9+16) = 5`

**单位向量**：长度为 1 的向量，只保留方向信息：

```python
v = np.array([3, 4])
v_hat = v / np.linalg.norm(v)
# v_hat = [0.6, 0.8]
# 验证：0.6² + 0.8² = 0.36 + 0.64 = 1.0 ✓
```

**为什么需要单位向量？** 当你想"往某个方向走"，你需要一个纯方向（长度 1），然后乘以步长。3DGS 中视线方向就是单位向量——只关心往哪看，不关心看多远。

## 3.3 点积 —— 两个向量的"对齐程度"

### 两个定义（数学上等价）

```
代数定义：a·b = a₁b₁ + a₂b₂ + a₃b₃      ← 代码里实际算的
几何定义：a·b = |a|·|b|·cos(θ)         ← 用来理解的
```

### 等价性证明（这是理解点积的关键）

**已知**：|a|, |b|, 夹角 θ

**用余弦定理**：
```
|c|² = |a|² + |b|² - 2|a||b|·cos(θ)     ← c = a - b
```

**用代数定义展开 |c|²**：
```
|c|² = (a-b)·(a-b)        ← 代数的点积
     = a·a - a·b - b·a + b·b
     = |a|² - 2(a·b) + |b|²
```

**对比两个表达式**：
```
|a|² + |b|² - 2|a||b|cos(θ)  =  |a|² - 2(a·b) + |b|²
                        -2|a||b|cos(θ)  =  -2(a·b)
                           |a||b|cos(θ)  =  a·b   ✓
```

**这个证明告诉你**：点积的两种定义不是巧合，是余弦定理的必然推论。

### 点积告诉你什么？

```python
a = np.array([3, 0])       # 指向正右方
b1 = np.array([2, 1])      # 右上，夹角小
b2 = np.array([0, 2])      # 正上，垂直
b3 = np.array([-2, 1])     # 左上，夹角大

print(np.dot(a, b1))   # 6    → 正数 = 锐角（< 90°）
print(np.dot(a, b2))   # 0    → 零 = 垂直（恰好 90°）
print(np.dot(a, b3))   # -6   → 负数 = 钝角（> 90°）
```

**判断规则**：

| 点积 | cos(θ) | θ 范围 | 含义 |
|------|--------|--------|------|
| > 0 | > 0 | 0° ~ 90° | 大致同向 |
| = 0 | = 0 | 恰好 90° | 垂直 |
| < 0 | < 0 | 90° ~ 180° | 大致反向 |

### 最重要的几何直觉：投影

> **a 在 b 方向上的投影长度 = `(a·b) / |b|`**

```python
a = np.array([3, 2])       # 任意方向
b = np.array([3, 0])       # x 轴方向
b_hat = b / np.linalg.norm(b)   # x 轴单位向量 = [1, 0]

proj_len = np.dot(a, b_hat)      # = 3×1 + 2×0 = 3
proj_vector = proj_len * b_hat   # = [3, 0]
```

**画在纸上**：a=(3,2) 是一个从原点出发指到 (3,2) 的箭头。把它投影到 x 轴（即 b 的方向），投影点就是 (3,0)。投影长度 = 3 恰等于 a 在 x 轴上的分量。

**这就是投影的全部**：`a·b̂` 算出"a 在 b 指向的方向上走了多远"。

### 3DGS 中怎么用点积？

**场景1：判断高斯在不在相机视野里**

相机的视线方向是 d（单位向量），从相机到高斯的向量是 v。如果 `d·v > 0`，说明高斯在相机前方；如果 `< 0`，在身后（扔掉不渲染）。

**场景2：球谐函数计算**

球谐函数本质是视线方向 d 和球谐基函数 `Y_lm` 的某种"方向相似度"计算，内积形式。

## 3.4 叉积 —— 造出一个垂直于两个向量的新向量

### 定义（仅限 3D 空间）

```python
a = np.array([2, 0, 0])    # x 轴方向
b = np.array([0, 2, 0])    # y 轴方向

c = np.cross(a, b)
print(c)   # [0, 0, 4]    ← z 轴方向，垂直于 xy 平面
```

### 手算公式（用行列式记忆）

```
a × b = det( | i   j   k |
             | a₁  a₂  a₃ |
             | b₁  b₂  b₃ | )

     = i(a₂b₃ - a₃b₂) - j(a₁b₃ - a₃b₁) + k(a₁b₂ - a₂b₁)
```

**一步一步算 a=(2,-1,3), b=(1,4,-2)**：

```
i 分量：a₂b₃ - a₃b₂ = (-1)×(-2) - 3×4 = 2 - 12 = -10
j 分量：a₁b₃ - a₃b₁ = 2×(-2) - 3×1 = -4 - 3 = -7  → 前面有负号 → +7
k 分量：a₁b₂ - a₂b₁ = 2×4 - (-1)×1 = 8 + 1 = 9

a × b = (-10, 7, 9)
```

### 验证：结果垂直于 a 和 b

```python
a = np.array([2, -1, 3])
b = np.array([1, 4, -2])
c = np.cross(a, b)

# 验证垂直性：两个垂直向量的点积 = 0
print(np.dot(c, a))   # -10×2 + 7×(-1) + 9×3 = -20-7+27 = 0 ✓
print(np.dot(c, b))   # -10×1 + 7×4 + 9×(-2) = -10+28-18 = 0 ✓
```

### 几何意义

**方向：右手定则**

右手食指 = a，中指 = b，拇指 = a×b。它永远指向 a 和 b 所在平面**的法向量**方向。

**大小：平行四边形面积**

```
|a × b| = |a|·|b|·sin(θ) = a 和 b 张成的平行四边形面积
```

验证 `a=(2,0,0), b=(0,2,0)`：|a|×|b|×sin(90°) = 2×2×1 = 4，而 `|c| = |(0,0,4)| = 4` ✓

### 关键性质

```python
# 1. 自己叉自己 = 零向量
np.cross(a, a)   # [0, 0, 0]

# 2. 反交换律：a×b = -(b×a)
np.allclose(np.cross(a, b), -np.cross(b, a))   # True
```

### 3DGS 中怎么用叉积？

**计算平面法向量**：建筑场景中，给定墙面上的三个点 p1, p2, p3，墙面的法向量方向 = `(p2-p1) × (p3-p1)`。这对于判断高斯椭球是否应该"躺平"贴在墙面上非常关键。

## 3.5 点积 vs 叉积 — 对比总结

```
           点积 a·b                    叉积 a×b
           ────────                   ────────
结果类型    标量（一个数）              向量（有方向有长度）
计算       a₁b₁ + a₂b₂ + a₃b₃         见行列式展开
几何意义   度量"对齐程度"              造法向量 + 算面积
方向信息   cos(θ) → 同向/垂直/反向     垂直于 a 和 b（右手定则）
等于零时   垂直                        平行或一个向量为零
3DGS用途   视野判断、球谐投影           法向量计算
```

# 3DGS科研项目协作规则 

## 环境约束 

- 运行环境：CUDA 11.8 + PyTorch 2.1.0 + Python 3.10 
- 代码规范：Python遵循PEP8，CUDA核函数必须加详细中文注释，标注输入输出、物理意义、并行逻辑
- 安全约束：执行rm、pip install、编译CUDA扩展、删除文件前，必须二次确认 

## 回答规范 

1. 讲解原理必须做到「数学公式 + 代码变量 + 物理意义」三者对应 
2. 所有公式统一使用 LaTeX 格式书写 
3. 涉及论文必须标注出处（作者、会议、年份） 
4. 分析代码先讲整体逻辑，再逐行拆解细节 
5. 实验结果自动整理为对比表格，标注关键结论

## 工作流程

- 接到任务先输出执行计划，确认后再动手
- 修改代码只做最小改动，不重写无关逻辑 
- 调试报错先定位根因，再给出修复方案和验证方法
